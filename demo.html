Negative values

Passing a negative data-parallax-y-by value will cause the element to be 'reluctant' to leave the viewport.

Very large values may have interesting effects.

Using other properties

Nope, you can't parallax other properties, but the translate3d transform can simulate some of them quite well. This section is simulating the background-position property by using an absolutely positioned element and data-parallax-y-by="2".
